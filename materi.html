Materi Fungsi Kubik

Definisi :

Fungsi kubik adalah fungsi polinomial derajat 3.
fungsi kubik juga ialah fungsi yang memiliki pangkat tertinggi adalah pangkat 3

Bentuk umum :

f(x) = ax3 + bx2 + cx + d.
di mana a , b , c , dan d adalah konstanta dan a tidak sama dengan nol

Bentuk kurva

Fungsi kubik ini memiliki bentuk kurva yang bisa naik atau turun tergantung nilai koefisiennya.

Ciri ciri fungsi kubik

Jika a > 0 grafik fungsi kubik akan turun dari kiri ke kanan, lalu naik kembali.
Jika a < 0 grafiknya akan naik dari kiri ke kanan, lalu turun kembali.
Contoh sederhana:
Jika f(x) = x ^ 3 grafiknya naik dari kiri ke kanan tanpa berbalik arah.
Jika f(x) = - x ^ 3 grafiknya turun dari kiri ke kanan.

Bentuk grafik fungsi kubik

Grafik fungsi kubik berbentuk kurva yang umumnya melengkung dan memiliki perubahan arah (baik naik-turun atau turun-naik).
Grafiknya bisa berpotongan dengan sumbu-x di tiga titik, dua titik, satu titik, atau bahkan tidak sama sekali.

Contoh Soal Fungsi Kubik

f(x) = x³-6x²+8x

•> Titik potong sumbu x (y=0)

y = x³-6x²+8x
0 = x (x²-6x+8)
0 = x (x-4) (x-2)

x1 = 0, -> (0, 0)
x2 = 4, -> (4, 0)
x3 =2, -> (2, 0)

•>Titik potong sumbu y (x=0)

y = x³-6x²+8x
y = (0)³ - 6(0)² +8(0)
y = 0 - 0 + 0
y = 0 -> (0,0)

•>Titik stasioner

y = x³-6x²+8x
y' = 3x²-12x+8

x1x2 = 12 ± 144- 96 6

x1x2 = 12 ± 6.92 6
x1 = 12 + 6.92 6
x1 = 3.15
